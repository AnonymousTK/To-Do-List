<b><h1>iOS-Style To-Do List 📝</h1></b>
A sleek, functional task management application designed to replicate the clean and intuitive user interface of the native iPhone Reminders app. This project focuses on minimalist design principles, fluid transitions, and a clutter-free user experience.
<br>
<b><h2>✨ Features:</h2></b>
<b>iOS Aesthetic:</b> Authentic replication of the iPhone UI, including rounded corners, subtle shadows, and system-style typography.

<b>Task Management:</b> Easily create, complete, and delete tasks with a single tap.

<b>Responsive Design:</b> Fully optimized for mobile and desktop browsers to maintain that "app-like" feel.

<b>Clean Interface:</b> Zero clutter—focusing entirely on your productivity
